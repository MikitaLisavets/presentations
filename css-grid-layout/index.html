CSS Grid layout

This CSS module defines a two-dimensional grid-based layout system, optimized for user interface design. In the grid layout model, the children of a grid container can be positioned into arbitrary slots in a flexible or fixed predefined layout grid.

History

  • 7 April 2011

				<div class="item_1">A</div>

			}
		
			.container {
				display: grid;
				grid-template-columns: repeat(3, 100px)
				grid-template-rows: repeat(2, auto);
			}
		
			.item_1 {
				grid-column-start: 1; // start grid line 
				grid-column-end: 2; // end grid line 
				grid-row-start: 1;
				grid-row-end: 2;
			}
		
			.item_2 {
				grid-column-start: 2;
				grid-column-end: 4;
				grid-row-start: 2;
				grid-row-end: 3;
			}
