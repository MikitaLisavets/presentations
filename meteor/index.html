Meteor is a full-stack JavaScript platform for developing modern web and mobile applications. Meteor includes a key set of technologies for building connected-client reactive applications, a build tool, and a curated set of packages from the Node.js and general JavaScript community.

Distributed Data Protocol

Distributed Data Protocol

Client-server protocol for querying and updating a server-side database and for synchronizing such updates among clients. It uses the publish-subscribe messaging pattern.

Latency compensation

Without Latency compensation

  1. +0ms: User clicks a Submit button and browser calls a method
  2. +200ms: Server updates MongoDB
  3. +500ms: Client gets updated data and render ui

With Latency compensation

  1. +0ms: User clicks a Submit button and browser calls a method
  2. +0ms: Client simulates the action method on their local collections and then render ui
  3. +200ms: Server updates MongoDB
  4. +500ms: Client gets updated data and synchronizes local database
